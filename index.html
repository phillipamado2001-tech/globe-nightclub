Google Sheet Setup

To manage events and all site content, create a Google Sheet with the required tabs.

  1. Create a Google Sheet with tabs: Events, Schedule, Bottles, VIP_Packages, Food, Gallery, Specials, SiteConfig
  2. Go to File → Share → Publish to web → Select "Entire Document" → Publish as CSV
  3. Copy the Sheet ID from the URL (the long string between /d/ and /edit)
  4. Paste it into js/config.js as the SHEET_ID value

Append ?refresh=1 to the URL after making changes to bust the cache.
